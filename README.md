# Insurance Fraud Detection Platform

End-to-end Data Engineering and Machine Learning platform built on Databricks.

## Architecture

- SQL Server (Claims Data)
- Amazon S3 (Claim Images)
- Amazon Kinesis (Telematics Streaming)
- Databricks (Bronze/Silver/Gold)
- MLflow
- Dashboard Layer

## Tech Stack

- Python
- PySpark
- Databricks
- Delta Lake
- AWS S3
- AWS Kinesis
- GitHub Actions
